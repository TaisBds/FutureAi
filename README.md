# 🚀 Future AI Lab

Este projeto foi desenvolvido como parte de uma iniciativa do **Porto Digital**, em parceria com a **Universidade Católica**, com foco no desenvolvimento de uma aplicação web utilizando tecnologias modernas de **Front-end e Back-end**.

---

## 🎯 Objetivo do Projeto

O objetivo do projeto é criar uma aplicação web moderna, com interface amigável e estrutura escalável, explorando conceitos reais utilizados no mercado.

---

## 🛠️ Tecnologias Utilizadas

- 🅰️ Angular  
- 🟦 TypeScript  
- 🎨 CSS  
- 🌐 HTML  
- 🟨 JavaScript  

---

## 📂 Estrutura do Projeto

- `src/` → Código principal da aplicação  
- `public/` → Arquivos públicos  
- `.vscode/` → Configurações do ambiente  
- `angular.json` → Configuração do Angular  
- `package.json` → Dependências do projeto  

---

## 🚀 Funcionalidades

- 🎨 Interface moderna e responsiva  
- 🔄 Atualização dinâmica da aplicação  
- 🧩 Componentização com Angular  
- 📁 Organização modular do projeto  

---

## ▶️ Como Executar o Projeto

### 📌 Pré-requisitos
- Node.js instalado  
- Angular CLI instalado  

### ▶️ Passos

```bash
npm install
ng serve
```

Acesse no navegador:

```
http://localhost:4200
```
